ThereQuickActions V3.0 Source
=============================

Windows version.

Changed
-------
- Auto Dance no longer shows the main Search bar.
- Follow Moves is moved higher in Auto Dance.
- Follow Moves has a saved avatar-name list.
- Add Name saves the typed name to the list.
- Remove deletes the selected saved name.
- Clicking a saved name fills the Follow Moves name box.
- Double-clicking a saved name follows it.
- Stop Follow now sends:
  /emote stop
